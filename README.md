# Beauty Theme for Soobu E-Commerce Platform

A beautiful, modern e-commerce theme designed for the Soobu platform. This theme is optimized for beauty, cosmetics, fashion, and lifestyle online stores.

## Features

- **Modern Design** - Clean and elegant UI with focus on product visualization
- **Fully Responsive** - Works seamlessly across all devices
- **Easy Customization** - Configure via theme settings panel
- **Fast Loading** - Optimized for performance
- **RTL Support** - Right-to-left language support

## Installation

1. Download or clone this repository
2. Upload to your Soobu themes directory
3. Activate through the theme customization panel

## Theme Structure

```
beauty/
├── assets/
│   ├── css/          # Stylesheets
│   ├── js/           # JavaScript files
│   ├── images/       # Theme images
│   └── webfonts/     # Font files
├── views/
│   ├── front_end/    # Frontend templates
│   └── components/   # Reusable components
├── theme_json/       # Theme configuration JSON files
├── default_data/     # Default settings
└── verification.php # Theme verification file
```
